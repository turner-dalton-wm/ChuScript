Overview

ChuScript is a programming language parsed and interpreted in Javascript. The goal of it was simplicity and the possibility of it being used to teach beginners basic principles in coding. As a result, there is only one data type in the language: numbers. These numbers can be floats or integers and have the same limitations in terms of size that the Number type variables in Javascript have.

There four keywords of this language are input, print, if, and function. The input keyword is similar to a prompt() in Javascript and is used to store user input in a variable. Print allows the user to output an expression to the console. An if statement resembles most C languages, but at the moment, is limited to the inline style without a block structure following it.


Functions

Functions in ChuScript have nearly identical syntax to functions in Javascript. They can be placed inside parent functions while still not being accessible outside the parent's scope.

The main difference between Javascript and ChuScript methods are how they return values. ChuScript does not use the return keyword like Java and Javascript, instead reference variables are used mainly because you can return multiple sets of data with one function call. Reference variables are noted in the function's parameters by a dollar sign in front of the variable name. If a reference variable's data is changed during the function's execution, it also changes the corresponding variable passed into the function as an argument.

Flexible Interpreter

In ChuScript, whitespace and line breaks do not matter to the interpreter. This allows for more options in your coding style as long as your brackets, semicolons and keywords are properly placed.
